# Task App

A simple task demonstration.

## Features
### Task 1

- Fetch products from the `dummyjson.com` API using pagination.
- Fetch about 40 products initially and implemented simulate pagination by showing 10 items at a time.
- Display the products in a scrollable list view.
- Product card with thumbnail image, title, category, price & rating.
- Loading indicator at the bottom while fetching more products.
- Product details screen with all give major details.


### Task 2

- Fetch all products from the `fakestoreapi.com` API.
- Display products in Sliver ListView for interactive list page.
- Product card with product image, title, price, category, & rating.

## Tech Stack
- Flutter & Dart
- Provider for state management
- Dio for API calls
- JSON parsing with custom models
